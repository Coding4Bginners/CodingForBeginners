Coding for Beginners is an (unauthorized) companion site for the New York Public Library's Project Code.

Point of entry

Open the Brackets application. Go to the File menu, pick New to make a new, blank document, and save it as index.html. As you save it, create a new folder; call that folder anything that will help you find it later. Maybe call it Week One, because this information is from week one of the NYPL's Project Code class.
